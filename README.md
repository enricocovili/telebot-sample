---

## Get it working:

### Standard way
```
git clone https://github.com/ginop-1/telebot-sample.git
cd telebot-sample
python3 -m venv venv
. venv/bin/activate
pip3 install -r requirements.txt
python3 src/main.py
```

### Docker
Docker image is included in the repo (note: ffmpeg will not be installed so youtube downloads won't work)
